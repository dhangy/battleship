Game Logic:

  1. A function that initializes the game object( which includes grids, ships etc). Must be called before game can be tested in console.

    initializeGame();

  2. Takes a random point and pass into a function that places a ship. Enter in what type of ship (carrier,battleship etc) then the row and column on the grid (the random point), then horizontal or vertical and lastly which grid you are placing it on (playerGrid or computerGrid)

    placeShip(ship, row, col, alignment, playerGrid);

  3. A function that checks to make sure that the ship placement function is going to be placed in a valid location. It gets called within the placeShip function to check the validity before the ship is ever placed

    GridCheck(row, col, size, alignment,playerGrid);

  4. A way to randomly generate points to be used on the grid

    generatePoint(); (returns a point)

  5. A hard coded ship placement used for testing. Comes with all ships being created in a set location. All you have to do is pass in which grid you are wanting to place on and whose ships you are placing: for example:

    testPlacement(playerGrid, playerShips)

    Or

    testPlacement(computerGrid, computerShips)

  6. A function that is used to launch missiles on the enemies grid. This function checks to see if the location you are passing in has already been shot at (hit or miss) if it has it notifies the user that it has already marked this point. If it hasnt then it checks the grid and marks it either hit or miss which are represented by O for miss and X for hit. If it hits it calls another function that marks the enemies Ship object as hit (markShipHit)

    launchMissile(row,col,grid, enemy)

    Inner function of launch Missile:

    markShipHit(turnShips,coordinates)

    This function loops over either the playerShips object or the computerShips object (depending on what you pass in the turnShips parameter) and checks each ships set location to see if they contain in the location array the same coordinates as what you passed in. When it finds it it incriments the hits attribute on that ship. It also checks to see if it sunk the ship that was hit. If so it increments the counter on the corresponding playerShips or computerShips shipsSunk. It also checks each time is sinks one to see if the game is over by calling the

    isGameOver()

    if it is it alerts the user.
  7. I am currently working on the computer guessing logic. Thus far I have made a function that does unintelligent random guessing. It takes a random point and calls the launch missile object on the playerGrid.

    computerRandomGuess()

  8. I am working on making the function

    targetComputerGuess()

    which will hopefully make it so that once the computer makes a hit on the following turn it will call this function which will pick an adjacent point to guess.
